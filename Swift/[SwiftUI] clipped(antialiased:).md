# [SwiftUI] clipped(antialiased:)

이 뷰를 bounding rectangular frame에 클립

도형의 레이아웃 bounds (경계) 를 벗어나는 내용을 숨길 수 있음

기본적으로 뷰의 경계 프레임 (bounding frame ) 은 레이아웃에만 사용되므로 프레임의 가장자리를 벗어난 내용은 계속 표시됨.
