# STON Edge Server

- 차세대 고성능 리버스 **프록시 서버**
- 시스템 아키텍쳐의 수정없이 **웹 서버 전단에 위치**하여 콘텐츠 **가속 전달** 그리고 기존 시스템과 **DB의 부하**를 획기적으로 절감
  - **캐싱** 파일시스템 (POSIX File I/O)
  - On-the-fly 이미지 최적화 및 변환
  - SSL/TLS 1.2 지원
  - **Load Balancing**
  - **트래픽, DB/스토리지 부하 절감**
  - 미디어 멀티프로토콜 지원
