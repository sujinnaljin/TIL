# GraphQL

- **GraphQL**은 현재 우리가 많이 사용하는 **RestAPI를 대체하는 기술로 API를 쿼리**하는 개념
- 기존 Rest API는 각각의 작업에 대해서 별도의 url, 즉 end point를 다 다르게 정의해야 하고, 스키마와 type 정의가 없기 때문에 이를 보완하기 위해 스웨거 같은 툴을 사용해야 함. 또한 요구 사항이 유사하더라도 입력 필드와 결과 필드의 디자인이 다르면 매번 새로운 api 와 endpoint 개발 필요. -> **GraphQL은 스키마와 타입을 기본적으로 지원하고 필요한 필드만 쿼리할 수 있도록 설계되어 네트워크 및 서버 자원 효율적**으로 사용 가능.
